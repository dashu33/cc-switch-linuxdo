# CC Switch v3.17.2-personal.1 更新说明（自用分支）

> 维护位置：`自用特性/发布说明-v3.17.2-自用版.md`

分支：`main`

版本：`3.17.2`

标签：`v3.17.2-personal.1`

## 本版更新

1. **NewAPI 全模型客户端导入**
   - NewAPI 统一供应商可同步到 Claude、Codex、Gemini、Grok Build、Claude Desktop、OpenCode、OpenClaw 和 Hermes。
   - 各应用按自身真实配置结构生成 Provider，不再只覆盖原有三个客户端。
   - 文档：`自用特性/NewAPI快速导入/`

2. **供应商收藏**
   - Provider 卡片左上角新增收藏星标。
   - 「仅收藏」快捷筛选位于「仅显示可用」旁边。
   - 收藏按应用分别保存在本机。
   - 文档：`自用特性/供应商收藏/`

3. **模型品牌图标选择最新旗舰模型**
   - 点击模型品牌图标时优先选择旗舰/通用型号。
   - 正确比较语义模型版本、档位、稳定性和发布日期。
   - 日期、上下文长度和参数量不再误作版本；轻量及专用型号会降级。
   - 文档：`自用特性/供应商行UI压缩与筛选/`

## 构建与发布

- Windows x64：`Personal Windows Release`
- macOS：`Personal macOS Release (unsigned)`，无 Apple Developer 签名 / 公证
- 全量签名多平台 `Release` 不属于本次发布，若被 tag 匹配则取消
- 本次按用户要求不处理本机安装版本
